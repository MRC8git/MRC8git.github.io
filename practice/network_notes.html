Network Notes

Protocol
The set of rules for communication between networking devices--how the information communicated should be formatted. One example is TCP/IP(Transmission Control Protocol/Internet Protocol)
Packet
A tiny piece of information, usually part of a larger message. A singular message could contain millions of packets.
Header
Basic information about a packet.(eg., To, From, order)
Payload
The message, the substance, of a set of packets.
Router
A device that connects devices and networks, that packets from a multitude of those devices which travels through.
Routing Table
A table that uses the IP adresses of routers to store their locations.
